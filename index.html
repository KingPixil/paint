Paint

Lightweight JS Library for DOM Manipulation/Events.

v1.0.1


What is Paint?

Let's face it, you don't need 100s of KB's of Javascript just to manipulate the DOM and see when a user does something. Paint aims to fix that for you. It comes with all of JQuery's core features that you need, and uses optimized code to make it blazing fast. If you've ever used JQuery, than learning how to use Paint will be no problem, it has a JQuery-like API, which is easy to understand. If not, then check out the docs below.

Open Source

Paint is 100% open source, so you can learn from it, and see what happens under the hood. Check it out on Github. Make sure to star it! Paint is also released under the MIT License, allowing you to use it in a variety of projects.

Lightweight

Paint is only ~1kb minified. Making it super fast to load and use. You don't need to compile any files, nothing complex. All you do is link to it and you are done. Because of Paint's lightweight file size, it will load much faster than <place-framework-here>

How you use Wing.

Get Started
Run npm install paintjs, and link to ./node_modules/paintjs/dist/paint.min.js
Also, you can just add this to your <head>: <link rel="stylesheet" href="http://usepaint.ml/dist/paint.min.js"/>
DOM Selection

Paint comes with simple DOM selection like JQuery out of the box. This means that you don't ever have to write those long statements just for getting access to a node, instead you can focus on the important stuff.

To use it, you can simply call the $ function with a cs selector inside declaring what element you want to manipulate.

          
$("#someID");
$("p");
$(".someClass")
$(".someClass p #idInPInClass");
Ready

Paint comes with a simple function that easily lets you know when the DOM is ready.

You simply call the .ready() function. This will attach an event to the document and will execute the callback function code when the DOM is ready. This prevents you from manipulating things in the HTML that haven't loaded yet. 

              
$().ready(function() {
  // Your code here
});
Each

Sometimes you need to do things with more than one Node (fancy word for an element). You have a NodeList. Paint comes with a simple function at its core that lets you do this. It will iterate through elements selected and execute the callback.

.each() comes in handy when you want to do something for a list of elements. For example, every H1 on the page. You can easily iterate through these and manipulate each one.

              
$("h1").each(function(node) {
  // manipulate the node
});
